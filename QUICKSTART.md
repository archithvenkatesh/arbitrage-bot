# Quick Start Guide

## Running the Bot

1. **Install dependencies** (first time only):
   ```bash
   npm install
   ```

2. **Start the server**:
   ```bash
   npm start
   ```

3. **Open your browser**:
   Navigate to http://localhost:3000

4. **Fetch live data**:
   Click "Refresh Data" or try "Demo Mode"

## Stopping the Server

Press `Ctrl+C` in the terminal where the server is running.

## That's it!

The bot is now running with live API integration. No CORS issues, no additional setup needed!
